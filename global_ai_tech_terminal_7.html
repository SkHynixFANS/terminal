How live data works in this terminal

All API keys are saved in your browser's localStorage and never leave your device.

Recommended setup:
1. Finnhub API Key (free) — finnhub.io — live real-time stock prices + news headlines. This is the primary price source (direct CORS, no proxy caching).
2. Yahoo Finance — No key needed. Used as price fallback + 2-year chart history (via CORS proxy).
3. Alpha Vantage (optional) — alphavantage.co — supplemental price history.

Without any keys: The terminal shows static seed prices and news from Yahoo/Google RSS feeds.
